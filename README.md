# Controle_Financeiro
Todos os dados dessa planilha são fictícios.

1. Controle - Uma comparação de lucro e gasto anual em um período de 5 anos.
2. Cotidiano - Lista de todas as contas fixas do mês + GPNP (Gastos Pontuais Não Previstos).
3. GPNP 1°S - Todos os seus gastos não previstos de janeiro a junho.
4. GPNP 2°S - Todos os seus gastos não previstos de julho a dezembro.
